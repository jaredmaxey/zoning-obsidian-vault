# CRE Knowledge Base — Master Index

A dual-purpose, AI-readable knowledge base: a **commercial real estate development brain** and a **jurisdiction-by-jurisdiction zoning reference** (Arizona built; more states to follow). This is the human entry point. AI sessions should start with **`CLAUDE.md`** instead.

> **Status:** content is `draft` and useful as orientation, but **not human-verified** — especially numeric zoning standards (all currently `TBD` + `needs-verification`). See [[Vault Conventions]] §7 for the status lifecycle.

[[CRE Brain Index]] — the full commercial real estate development knowledge half.

- **Concepts** ([[CRE Concepts Index]]) — cap rate, NOI, IRR, DSCR, LTV/LTC, yield on cost, and 19 more metrics.
- **Deal Types** ([[Deal Types Index]]) — core through opportunistic, plus sale-leaseback, build-to-suit, land banking, etc.
