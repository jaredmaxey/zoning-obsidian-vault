# Case Studies Index

## Overview
Anonymized deal write-ups live in this folder. Each case study is a single note created from `_templates/template-case-study.md` and named `YYYY-MM-DD-{slug}.md` (the date is the write-up date, not necessarily the deal date).

Case studies are **not fabricated**. A note is only added here when there is a real (anonymized) or explicitly hypothetical deal to document — and any teaching example must set `outcome: hypothetical` in frontmatter so it is never mistaken for a realized track record.
