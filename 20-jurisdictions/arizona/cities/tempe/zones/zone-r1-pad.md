# Tempe R1-PAD

## Summary

The R1-PAD (Single-Family Planned Area Development) District is a flexible single-family residential district in which dimensional standards are established by the approved PAD (Planned Area Development) overlay and development plan rather than by fixed table values. The ZDC Table 4-202A shows NS (No Standard) for all dimensional rows — density, min lot area, lot width, lot length, height, lot coverage, and all setbacks. Each R1-PAD project must have a recorded development plan that sets its specific standards.

Standards transcribed from the official Tempe ZDC (Table 4-202A), provided 2026-05-25; spot-check recommended before promoting to authoritative.

## Permitted Uses

Per approved PAD development plan. Verify against ZDC use table (Part 3, Land Use) and the specific PAD document.
