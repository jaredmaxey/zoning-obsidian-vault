# Tempe AG

## Summary

The AG (Agricultural) District is Tempe's lowest-intensity base zone, intended to accommodate agricultural uses and large-lot single-family residential development where agricultural character is to be preserved. Because Tempe is a fully built-out, landlocked city, this district is rarely encountered in active development contexts; where it exists it typically represents legacy parcels or large institutional landholdings that have not yet been rezoned to a more intensive designation. Redevelopment of AG-zoned parcels typically requires a rezoning to a more appropriate base district.

Standards transcribed from the official Tempe ZDC (Table 4-202A), provided 2026-05-25; spot-check recommended before promoting to authoritative.
