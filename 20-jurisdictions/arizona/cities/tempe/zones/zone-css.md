# Tempe CSS

## Summary

The CSS (Commercial Shopping/Service) District is the primary general commercial base zone in Tempe, accommodating retail, service, food-and-beverage, auto-oriented commercial, and hospitality uses. Residential use at up to 20 du/acre is permitted by use permit (U). CSS was formerly known as CCR, C-1, and C-2 districts. Development standards are governed by ZDC Part 4, Table 4-203A.

Standards transcribed from the official Tempe ZDC (Table 4-203A), provided 2026-05-25; spot-check recommended before promoting to authoritative. Prior web-sourced height of "35" is confirmed correct.
