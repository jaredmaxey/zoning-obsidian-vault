# Tucson Overlay Districts (Index)

## Summary

Tucson's Unified Development Code contains overlay districts that modify base zone standards for specific geographic areas, land features, or policy objectives. This index is **preliminary** — the full overlay roster requires enumeration directly from the current Tucson UDC overlay article. The notes below represent overlays that are reasonably well-established based on general knowledge of Tucson's regulatory framework; additional overlays (scenic corridor, environmental resource, neighborhood commercial, etc.) may exist and are flagged as TBD (`needs-verification`).

Overlay districts layer on top of base zone districts. A parcel within an overlay must comply with both the base zone standards and the overlay's modifications, which may add requirements, relax requirements, impose design review, or change the approval process.

## Modifications to Base Zoning

Modifications vary by overlay type — see individual overlay notes for details. In general: Historic Preservation Zones impose demolition controls and design review; Rio Nuevo provides incentives and modified development standards; hillside overlays restrict grading and building height on slopes; scenic corridor overlays limit sign height and area; environmental resource overlays restrict development within riparian buffers.

## Additional Requirements

Process changes vary: HPZ requires Historic Preservation Office review; Rio Nuevo involves the Rio Nuevo Board; certain overlays require specific board or commission sign-off in addition to the standard PDS review process.

## Process Changes

See individual overlay notes for process modifications. As a general matter, overlay districts add a layer of review — design review, board approval, or enhanced staff scrutiny — on top of the standard Development Package process. Overlay applicability should be confirmed at the pre-application stage.
