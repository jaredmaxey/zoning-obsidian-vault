# Gilbert Overlay Districts (Index)

## Summary

The Town of Gilbert's Land Development Code contains one or more overlay districts that impose additional standards or modified requirements on top of the underlying base zones within specific geographic areas. This index note catalogues the known and candidate overlay districts; the full and authoritative roster must be confirmed against the current Gilbert LDC (needs-verification).

Overlay districts in Gilbert are designed to address special circumstances — preservation of the historic downtown character (Heritage District), promotion of high-quality employment development along major corridors, and potentially agritourism or heritage agricultural uses in areas that reflect the town's agricultural roots.
