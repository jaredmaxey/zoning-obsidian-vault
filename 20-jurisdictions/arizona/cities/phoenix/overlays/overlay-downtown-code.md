# Phoenix Downtown Code

## Summary

The Phoenix Downtown Code is a form-based or specialized urban overlay applicable to Downtown Phoenix, replacing or supplementing the standard base-zone regulations with standards tailored to the walkable, high-density, mixed-use urban core. Rather than regulating primarily through use lists and dimensional limits, the Downtown Code emphasizes building form, street activation, ground-floor use requirements, and public realm design to shape a pedestrian-friendly downtown environment consistent with Phoenix's urban revitalization goals.

The Downtown Code functions as a specialized regulatory framework that coordinates with — and in many respects overrides — the underlying base zone within its geographic boundary. It is closely related to Phoenix's broader downtown planning priorities, including the Downtown Phoenix Master Plan, streetscape standards, and transit-oriented development policies.
