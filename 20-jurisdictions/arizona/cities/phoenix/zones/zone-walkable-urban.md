# Phoenix Walkable Urban (WU) Code

## Summary

The Phoenix Walkable Urban (WU) Code is a form-based, transect-based regulatory framework that represents a significant departure from Phoenix's traditional Euclidean base-zone system. Rather than organizing land use by use category (residential, commercial, industrial), the WU Code organizes development by physical form and urban context — a "transect" approach that moves from lower-intensity, more suburban or neighborhood-scale contexts to higher-intensity, urban core contexts, with each transect zone specifying the permissible building types, heights, frontage treatments, and use mixes appropriate to that context.

The WU Code was adopted by the City of Phoenix to promote walkable, pedestrian-oriented, mixed-use development along the Valley Metro light-rail network and in designated urban activity centers. It is designed to facilitate the kind of transit-supportive, mixed-use, higher-density development that the Phoenix General Plan ("PHX General Plan") envisions for light-rail corridors — where reduced parking requirements, active ground-floor uses, and buildings built to the sidewalk are desired outcomes.

## Framework Characteristics

- **Form-based regulation**: standards govern building form (height, frontage type, stepbacks, ground-floor activation) rather than just use. The intent is to ensure urban-quality street environments regardless of the specific land use above the ground floor.
- **Transect zones**: the WU Code employs a series of transect districts progressing from less to more intense urban contexts. Specific WU transect district codes and standards: TBD — verify against current Walkable Urban Code. `needs-verification`.
- **Light-rail corridor application**: the WU Code is primarily applied within walking distance of Valley Metro light-rail stations, along key corridors, and in designated activity centers where the General Plan supports increased density and walkability.
- **Parking reductions**: the WU Code typically allows reduced or eliminated minimum parking requirements in transit-proximate locations, consistent with TOD policy goals.
- **Ground-floor activation**: certain transect zones require active uses (retail, restaurant, service) at the ground floor along primary street frontages, creating the kind of pedestrian environment that supports transit ridership and neighborhood vitality.

## Geographic Application

The WU Code applies in specific geographic areas designated through a WU rezoning process, not citywide. Application areas are concentrated near Valley Metro light-rail stations and along select arterial corridors. The precise boundaries and applicable transect designations for each area must be verified against the current Phoenix Zoning Ordinance WU Code maps and the PDD website.

> **Needs-verification**: Specific WU transect district codes (e.g., T3, T4, T5, T6 or Phoenix-specific designations), their standards, geographic coverage maps, and current adoption/amendment history — verify against current Walkable Urban Code documentation at https://www.phoenix.gov/pdd/zoning.

## Relationship to Euclidean Base Zones

In areas where the WU Code applies, it replaces or supplements the underlying Euclidean base zone. The relationship between the WU Code and the base zone (whether the WU Code functions as a replacement zone or an overlay) should be verified against the current Phoenix Zoning Ordinance. Where the Downtown Code and TOD Overlay also apply, the interaction between these frameworks adds additional complexity — verify applicable standards with PDD.
